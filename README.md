## Info_ip

Auteur : Antoine Even  
Nom : Info_ip

Il s'agit d'un script Bash qui permet d'avoir plusieurs types d'informations sur le système
dont l'ip publique et l'ip locale. 
Requiert les droits d'administrations.
ln-sensors & curl doivent être installés.

Utilisation :
`sudo ./info_ip.sh`
En root : 
`./info_ip.sh`
